## 5. Project instruction
### Step 1: Clone the repository

1. Open a terminal window.
2. Run the following command to clone the project repository:

    ```bash
    git clone https://github.com/RMIT-Vietnam-Teaching/group-project-cosc3060-cosc3061-2024b-g2
    ```

### Step 2: Navigate into the project directory

1. Change directory to the project folder by running:

    ```bash
    cd group-project-cosc3060-cosc3061-2024b-g2
    ```

### Step 3: Install the dependencies

1. Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
2. Run the following command to install all the necessary dependencies:

    ```bash
    npm install
    ```


### Step 4: Start the application

1. Run the following command to start the server:

    ```bash
    npm start
    ```

2. The frontend server should now be running on `http://localhost:3000`.
    The backendd server should now be running on `http://localhost:8800`.
### Step 5: Access the website

1. Open a web browser and go to:

    ```bash
    http://localhost:3000
    ```

2. You should see the website running locally.
